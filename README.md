
# ISTExpert.ru — Платформа контроля автопарка

Профессиональная система управления парком спецтехники с неоморфным дизайном (Soft UI).

## Основные возможности
- **Учет наработки**: Автоматический расчет моточасов.
- **Планирование ТО**: Генерация регламентов с помощью Gemini AI.
- **Снабжение**: Kanban-доска для закупок запчастей.
- **QR-Паспорта**: Мгновенный доступ к истории каждой единицы техники.

## Стек технологий
- React 19 + TypeScript
- Vite
- Zustand
- Tailwind CSS
- Gemini API

## Установка
1. `npm install`
2. Создайте файл `.env` и добавьте `API_KEY=ваш_ключ_gemini`
3. `npm run dev`
