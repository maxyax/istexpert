
# ISTExpert.ru — Система управления автопарком

Профессиональная платформа для управления парком автомобилей и спецтехники с современным дизайном.

## Основные возможности
- **QR-Паспорта**: Мгновенный доступ к истории каждой единицы техники
- **Учет наработки**: Автоматический расчет моточасов и интервалов ТО
- **Планирование ТО**: Автоматические регламенты и календарь обслуживания
- **Снабжение**: Kanban-доска для закупок запчастей
- **Документы**: Электронные паспорта, страховки, акты
- **Аналитика**: Статистика расходов и простоев

## Стек технологий
- React 19 + TypeScript
- Vite
- Zustand
- Tailwind CSS
- Lucide React (иконки)
- QRCode генерация

## Установка
1. `npm install`
2. Создайте файл `.env` и добавьте необходимые переменные окружения
3. `npm run dev`
